# Code of Conduct

## Our pledge

The BT Bridge Templates project is committed to providing a welcoming, harassment-free
experience for everyone who participates — contributors, maintainers, and users alike —
regardless of background or identity.

## Our standard

This project adopts the **Contributor Covenant, version 2.1**, as its Code of Conduct.
The full, authoritative text is published at:

> https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project (opening issues, submitting pull requests, or engaging in
discussions), you agree to uphold that standard. In short: be respectful, assume good faith,
welcome newcomers, and accept constructive feedback gracefully.

## Scope

This Code of Conduct applies within all project spaces — the repository, issues, pull requests,
discussions — and in any public space when an individual is representing the project.

## Reporting

If you experience or witness unacceptable behavior, report it confidentially to the project
maintainers at:

> **conduct@coldboreballisticsllc.com**

All reports will be reviewed and investigated promptly and fairly. Maintainers are obligated to
respect the privacy and security of the reporter of any incident.

## Enforcement

Maintainers are responsible for clarifying and enforcing this Code of Conduct and will take
appropriate, fair corrective action in response to any behavior they deem inappropriate. The
enforcement guidelines follow those described in the Contributor Covenant v2.1 (see the link
above).
